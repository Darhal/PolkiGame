## Introduction 
TrikytaEngine is lightweight 2D game engine created in C++ using SDL library and still under development stage.

## Current Features
Currently the engine features are limited to:
!(Some are not fully implmented yet)!
* Sprites creation
* Animations
* Engine Basics
* Inputs
* Physics
* Tiled Maps
* Events
